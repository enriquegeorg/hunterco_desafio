O propósito: ser uma alternativa rápida para lançamento de pedidos das salsichas do Sr. Bouthin (https://www.instagram.com/srbouthin/?hl=pt-br). Visa agilizar o recebimento dos pedidos e evitar perda de dados dos pedidos. A solução SERÁ EXTREMAMENTE SIMPLES, não emitira nf, não terá grandes api's (fique a vontade para embutir).

O desafio: usando a base-code do outro repositório "desafiohunterco" alterar o projeto mantendo o mesmo design-pattern.
